AdminPostersAdapter

open class AdminPostersAdapter

RecyclerView adapter used on the Admin Dashboard and Admin Images screen to display event poster thumbnails.

This adapter binds each Event's poster URL and title into a compact visual layout. Glide is used for efficient image loading and caching. The adapter exposes a click callback through OnPosterClickListener so the parent screen can respond to poster taps (e.g., open preview or navigate to details).

This class is a thin wrapper around ListAdapter, using a diffing strategy based on event IDs to efficiently update the UI.
