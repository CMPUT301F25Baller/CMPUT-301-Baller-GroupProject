NotificationAdapter

open class NotificationAdapter

RecyclerView adapter for displaying notification rows. Each item represents a single notification with a sender, message content, timestamp, and optional action buttons (e.g., "Mark as Read" and "Open").

This adapter uses a simple internal list and does not use DiffUtil, as most notification displays do not require complex item animations.
