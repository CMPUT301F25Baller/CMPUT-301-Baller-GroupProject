FirestoreEventRepository

open class FirestoreEventRepository

Repository class that provides Firestore-based data operations for Event objects. This class handles common queries such as trending events, organizer-filtered events, real-time listeners, and fetching individual event documents.

All methods use callbacks for asynchronous operations and return ListenerRegistration where applicable for real-time updates.
