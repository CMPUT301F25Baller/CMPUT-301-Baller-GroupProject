EntrantMainActivity

open class EntrantMainActivity

Main home screen for users with the entrant role.

This activity is responsible for:

  • Showing horizontally scrolling lists of trending and "near you" events.
  • Providing a search bar and category chips to filter all events.
  • Listening in real-time to the events collection in Firestore.
  • Navigating to DetailsActivity when an event card is tapped.
  • Navigating to ProfileActivity from the top-left menu button.

Data Flow:

  1. loadAllEvents attaches a Firestore snapshot listener on events.
  2. All events are cached in allEvents for client-side search/filter.
  3. Trending vs Near-You separation is based on isTrending.
  4. performSearchAndFilter delegates filtering logic to EventFilter.performSearchAndFilter.
