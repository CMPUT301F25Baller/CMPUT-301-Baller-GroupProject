AdminImagesAdapter

open class AdminImagesAdapter

Adapter for displaying event posters in the Admin Images gallery. This allows the administrator to browse images uploaded by organizers and remove them if necessary[cite: 105, 108].

User interactions:

  • Click: Previews the poster in a dialog.
  • Long-press: Prompts to delete (clear) the poster URL.
