NotificationLog

open class NotificationLog

Simple model representing a notification log entry, typically used for displaying past or archived notifications in a list.

Each entry includes an identifier, a title or message label, timestamp, avatar resource, and read/unread state. This model is UI-focused and does not integrate directly with Firestore real-time updates in the same way the core Notification model does.
